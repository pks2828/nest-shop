# Teslo API

1. Clonar proyecto
2. Npm install
3. Clonar archivo ```.env.template``` y renombrar la copia a ```.env```
4. Cambiar variables de entorno
5. Levantar base de datos
```
docker-compose up -d
```
6. Levantar: ```npm start:dev```
